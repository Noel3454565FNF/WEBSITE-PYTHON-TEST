How can this happend :

The Reactor can enter in a Unstable State if :

→ Reactor Temperature too hot for a long time.

→ Reactor Temperature too cold for a long time.

→ Reactor SafeGuard offline for a long time.

→ Reactor Stabilization/Power/Cooling Lasers Error.

→ Reactor Stabilization/Power/Cooling Lasers Implosion.

→ Reactor Stabilization/Power/Cooling Lasers Explosion.

→ Reactor Internal pressure too high.

→ Unauthorized biological form in the Reactor Ventilation.
